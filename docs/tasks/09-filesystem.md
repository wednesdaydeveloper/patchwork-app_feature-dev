# #09 ファイルシステムユーティリティ

- **ステータス**: `[x]` 完了
- **フェーズ**: 2. 型定義・データ層
- **前提**: #02

## 概要

`utils/fileSystem.ts` で布地画像のドキュメントディレクトリ保存／削除／パス解決を実装する。

## 受け入れ条件 (AC)

- [x] 画像 URI を受け取り、アプリのドキュメント領域へコピーできる
- [x] 削除時に物理ファイルも削除される
- [x] パスは `expo-file-system` の正規化パスを返す
- [x] 保存先ディレクトリが存在しなければ作成

## メモ

- `expo-file-system` の File / Directory / Paths API を利用。
- ファイル名の衝突回避（UUID やタイムスタンプ付与）。
