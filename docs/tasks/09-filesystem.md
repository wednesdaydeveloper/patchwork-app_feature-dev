# #09 ファイルシステムユーティリティ

- **ステータス**: `[ ]` 未着手
- **フェーズ**: 2. 型定義・データ層
- **前提**: #02

## 概要

`utils/fileSystem.ts` で布地画像のドキュメントディレクトリ保存／削除／パス解決を実装する。

## 受け入れ条件 (AC)

- [ ] 画像 URI を受け取り、アプリのドキュメント領域へコピーできる
- [ ] 削除時に物理ファイルも削除される
- [ ] パスは `expo-file-system` の正規化パスを返す
- [ ] 保存先ディレクトリが存在しなければ作成

## メモ

- `expo-file-system` の File / Directory / Paths API を利用。
- ファイル名の衝突回避（UUID やタイムスタンプ付与）。
