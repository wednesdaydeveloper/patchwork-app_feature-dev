## 概要

タブレット表示時、残りの画面(エクスポート / 設定 / 布地登録キャリブ / サイズ選択 / 各種ダイアログ)を画面幅を活かした見やすいレイアウトに整える。

## 受け入れ条件 (AC)

- [ ] エクスポート画面: 左にプレビュー、右にオプション(画像形式 / 用紙サイズ / アクション)を配置
- [ ] 設定画面: 中央寄せ、最大幅 600 DP 程度に制約
- [ ] サイズ選択画面(`/new-work/size`): 中央寄せ、最大幅制約
- [ ] CalibrationScreen: ルーラを画面幅に合わせて延長(現状 150mm 固定 → タブレットでは 200mm 等)

- [ ] 既存テストグリーン
