# #20 トースト・ダイアログのグローバル制御

- **ステータス**: `[ ]` 未着手
- **フェーズ**: 5. 共通 UI コンポーネント
- **前提**: #19, #06

## 概要

`atoms/notification.ts` でトースト・ダイアログ表示状態を管理し、ルートに `<NotificationHost />` を配置する。

## 受け入れ条件 (AC)

- [ ] 任意の場所から `showToast(...)` / `showDialog(...)` で呼び出せる
- [ ] エラー種別（CLAUDE.md「エラーハンドリング方針」）に対応
- [ ] 複数同時表示をキューで処理

## メモ

- ダイアログは Modal で実装。
- トーストは画面下部に表示、自動クローズ時間を指定可能にする。
