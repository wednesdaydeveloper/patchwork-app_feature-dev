# #01 Expo プロジェクト初期化

- **ステータス**: `[ ]` 未着手
- **フェーズ**: 1. プロジェクト基盤セットアップ
- **前提**: なし

## 概要

`create-expo-app` で TypeScript テンプレートを使ってプロジェクトを初期化する。

## 受け入れ条件 (AC)

- [ ] `npx expo start` で開発サーバが起動する
- [ ] `tsconfig.json` で `strict: true` が有効
- [ ] iOS / Android シミュレータで起動確認できる

## メモ

- コマンド: `npx create-expo-app@latest --template blank-typescript`
