## 概要

選択中ピースの画像対応づけを UI から解除できるようにする。`removePieceSettingAtom` は既存。

## 受け入れ条件 (AC)

- [x] 選択中ピースに割り当てがある場合、「対応づけを解除」ボタンを表示
- [x] 押下で確認なしに即時解除（履歴に積むので Undo で戻せる）
- [x] 解除後はピースが未割り当ての白塗り表示に戻る
- [x] Undo / Redo で復元できる
