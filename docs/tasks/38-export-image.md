# #38 画像エクスポート（PNG / JPEG）

- **ステータス**: `[ ]` 未着手
- **フェーズ**: 10. エクスポート機能
- **前提**: #37

## 概要

`react-native-view-shot` でキャンバスをキャプチャし、カメラロールに保存する。

## 受け入れ条件 (AC)

- [ ] PNG（透明背景対応）または JPEG で出力
- [ ] 長辺 1080px 以上の高解像度
- [ ] パーミッション拒否時にメッセージ表示
- [ ] 失敗時は再試行ボタン付きトースト

## メモ

- 保存には `expo-media-library` を利用。
- 高解像度キャプチャ用にプレビューと別の高解像度ビューをオフスクリーンで描画する。
