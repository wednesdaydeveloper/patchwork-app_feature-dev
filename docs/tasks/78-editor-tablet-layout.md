# #78 編集画面の横並びレイアウト(タブレット)

- **ステータス**: `[x]` 完了
- **フェーズ**: 17. タブレット対応
- **前提**: #76, #27, #29
- **重要度**: 🔴 効果大

## 概要

タブレット横向き時、編集画面のレイアウトを縦積み(キャンバス + 下部の `FabricPicker`)から **左右分割** に変更する。左側に大きなキャンバス + 調整コントロール、右側に布地ピッカーを配置。

## 受け入れ条件 (AC)

- [x] phone: 既存の縦積みレイアウト(変更なし)
- [x] tablet 横向き: 左右分割
  - 左カラム: ヘッダ(ワーク名 + サイズ + ツールバー) + キャンバス + 調整 / 解除ボタン
  - 右カラム: `FabricPicker`(布地一覧)
- [x] キャンバスサイズは左カラム幅と画面高さの小さい方を最大とし、サイズ感を改善
- [x] 調整モード時は右カラムを非表示(現状の挙動を継承)
- [x] AdjustOverlay の表示崩れがない
