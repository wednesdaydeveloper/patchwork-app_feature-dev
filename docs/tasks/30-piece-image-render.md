# #30 ピース内画像描画（クリッピング）

- **ステータス**: `[ ]` 未着手
- **フェーズ**: 9. パッチワーク作成・編集画面
- **前提**: #27, #29

## 概要

SVG `clipPath` でピース形状にクリップして画像を描画する。

## 受け入れ条件 (AC)

- [ ] 画像がピース形状にクリップされる
- [ ] 初期値は「ピースに収まる最小倍率」で計算
- [ ] 複数ピースで独立して描画される

## メモ

- `react-native-svg` の `Image` + `ClipPath` を利用。
- 初期倍率の計算はピースのバウンディングボックスと画像サイズから求める。
