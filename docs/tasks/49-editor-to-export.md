# #49 エクスポート画面への導線

- **ステータス**: `[x]` 完了
- **フェーズ**: 13. UX 追加機能
- **前提**: #35, #37

## 概要

編集画面とホーム画面からエクスポート画面（`/export/[id]`）に遷移するためのボタンを配置する。

## 受け入れ条件 (AC)

- [x] 編集画面のツールバーに「エクスポート」ボタン
- [x] 未保存の作品ではエクスポートできないため、保存済み（`editingWorkId !== null`）でない場合は disabled、または保存を促す
- [x] ホーム画面の作品一覧アイテムに長押し以外の補助メニュー（︙）でエクスポート選択肢を出す（任意）
- [x] 押下で `/export/[id]` へ遷移

## メモ

- 編集画面では `editingWorkId` を URL に渡す。
- ホーム画面の補助メニューは UX 検討事項。シンプルに編集画面経由のみでも可。
