## 概要

登録済み布地の名前・カテゴリを後から編集できるようにする。

## 受け入れ条件 (AC)

- [x] 布地一覧アイテムをタップで編集ダイアログ（`PromptDialog`）を表示
- [x] 名前・カテゴリを変更し DB に反映
- [x] 一覧の表示順・カテゴリ別グルーピングが即時更新される
- [x] 既存パッチワークから参照されていても影響なし（id は維持）

## メモ

- DB ユーティリティに `updateFabric(fabric: FabricImage)` を追加（既存 `insertFabric` の隣）。
- atom 側にも `updateFabricAtom`（`fabricsAtom` を差し替え）を追加。
- 削除と編集の操作差別化が必要：
  - タップ → 編集
  - 長押し → 削除
  - 既存 UI と一致させる。
