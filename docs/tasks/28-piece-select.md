# #28 ピース選択

- **ステータス**: `[x]` 完了
- **フェーズ**: 9. パッチワーク作成・編集画面
- **前提**: #27, #13

## 概要

タップでピースを選択し、`selectedPolygonIdAtom` を更新する。選択中ピースをハイライト表示する。

## 受け入れ条件 (AC)

- [x] タップ判定が正しいピースで反応する（凹多角形・曲線含むピースに対応）
- [x] 選択中ピースが視覚的に区別できる
- [x] 領域外タップで選択解除

## メモ

- ピース形状は SVG path（直線・曲線）。実装方針:
  1. path をサンプリングして近似ポリラインに変換
  2. 近似ポリラインに対して **Ray casting** で内外判定
- 重なり順は描画順の逆（最前面から判定）。
